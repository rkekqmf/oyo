# oyo 전체 레이아웃

## 탭 구성 (상단 탭)

| 탭 | 용도 | 비고 |
|----|------|------|
| **캐릭터 검색** | 검색 폼, 최근검색, 즐겨찾기, 메인 캐릭터 패널, 원정대 결과, 캐릭터 비교 | 기존 유지 |
| **경매장 추이** | 시세·트렌드, 관심 아이템 | 기존 유지 |
| **보스 준비도** | 캐릭터별 보스 입장 가능 여부 | 기존 유지 |
| **점수·랭킹** | 로펙 스타일 환산 점수, 레벨 구간 중앙값, 티어컷 | 신규 · 계산 로직 연동 예정 |
| **수익·효율** | 껨산기 스타일 컨텐츠 보상·시급, 환율, 재료 효율 | 신규 · 계산 로직 연동 예정 |
| **전투·계산** | LOALAB 스타일 치명타, 재련, 경매 계산기 등 | 신규 · 계산 로직 연동 예정 |

## 계산 로직 담당

- **점수·랭킹**: 환산 점수식, 중앙값/티어컷 데이터 → `utils/scoreRank.js` 등
- **수익·효율**: 보상표, 환율, 시급 계산 → `utils/revenueEfficiency.js` 등
- **전투·계산**: 치명타, 재련 비용/확률, 경매 낙찰 등 → `utils/combatCalc.js` 등

UI/레이아웃은 그대로 두고, 위 유틸만 채워 넣으면 각 패널에 연동 가능.

## 컴포넌트

- `ScoreRankPanel.jsx` — 점수·랭킹
- `RevenueEfficiencyPanel.jsx` — 수익·효율
- `CombatCalcPanel.jsx` — 전투·계산

현재는 플레이스홀더 문구만 노출. 계산 모듈 구현 후 각 패널에서 import 해서 사용.
